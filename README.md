# Weather Dashboard

## Description

Weather Dashboard is a responsive web application that allows users to search for current weather conditions and a 5-day forecast for cities around the world. The application retrieves real-time weather data from a server-side API and displays useful information such as temperature, humidity, wind speed, and weather conditions.

This project was built as part of a collaborative group assignment to simulate a real-world development environment using Git, GitHub, and agile development practices.

Users can search for a city, view current weather data, and access previously searched cities through saved search history.

## Features

* Search for weather by city name
* View **current weather conditions**
* View **5-day weather forecast**
* Display temperature, humidity, wind speed, and weather icons
* Save search history using **localStorage**
* Click previous searches to reload weather data
* Responsive layout for desktop, tablet, and mobile devices
* Clean and modern UI built with Bootstrap
* Uses modals for error messages

AS A traveler
I WANT to search for the weather in a city

---

## Acceptance Criteria

* GIVEN a weather dashboard with a search form

* WHEN I search for a city

* THEN I am presented with current and future weather conditions for that city

* WHEN I view the current weather conditions

* THEN I see the city name, date, temperature, humidity, wind speed, and weather icon

* WHEN I view the future weather conditions

* THEN I am presented with a 5-day forecast

* WHEN I search for a city

* THEN that city is added to search history

* WHEN I click a city in the search history

* THEN I am presented with current and future conditions for that city again
